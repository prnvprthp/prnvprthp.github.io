- **GitHub Pages Ready**: Optimized for static export and automated deployment.

## Development

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment to GitHub Pages

This project is configured to deploy automatically via GitHub Actions.

1.  **Push your changes**:
    ```bash
    git push origin main
    ```
2.  **Enable GitHub Actions Deployment**:
    - Go to your repository on GitHub.
    - Navigate to **Settings** > **Pages**.
    - Under **Build and deployment** > **Source**, change it from "Deploy from a branch" to **GitHub Actions**.
3.  **Wait for the Build**: Your site will be live at `https://prnvprthp.github.io/` shortly after the Action finishes.
